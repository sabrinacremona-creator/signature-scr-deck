
# Signature Costa Realty — Deck Web

Single-page React deck (Vite + Tailwind). Design: luxe discret, couleurs de marque.

## Local

```bash
npm install
npm run dev
# open http://localhost:5173
npm run build
npm run preview
```

## GitHub (exemple)

```bash
git init
git add .
git commit -m "init: Signature Costa Realty deck"
git branch -M main
git remote add origin YOUR_GITHUB_REPO_URL
git push -u origin main
```

## Déploiement rapide

- **Vercel** : importez le dépôt, framework = Vite.
- **Netlify** : build command `npm run build`, publish dir `dist`.
- **OVH** (hébergement statique) : uploadez le contenu de `dist/` via FTP.
